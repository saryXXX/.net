# Dev 2 — Frontend (Blazor UI/UX)

## Responsabilité
Expérience utilisateur + CRUD.

## Missions
- **Pages Blazor** :
  - Clients
  - Produits
  - Factures
  - Stock
- **Formulaires avec validation**.
- **Composants réutilisables**.
- **Navigation + Layout** (incluant routes protégées).
- **Gestion UI Sécurité** : Login, Logout, Profil.
- **Feedback Erreurs UX** : Notifications et états d'erreur.

## Livrables
- UI fluide.
- CRUD complet.
- Gestion des formulaires.
- UX propre.
- Flux d'authentification complet.
- Gestion élégante des erreurs API.

## Organisation Agile (Sprints)

### Sprint 0 — Setup
- Navigation.
- **Pages Login / Register**.
- **Intercepteurs d'erreurs (401, 500)**.

### Sprint 1 — Core CRUD
- UI Clients.
- UI Produits.
- **Filtres d'affichage (actifs vs supprimés)**.

### Sprint 2 — Facturation
- UI Facture.
- Ajout produits dynamique.

### Sprint 3 — Stock
- UI Stock.
- UI Mouvements.

### Sprint 4 — Dashboard
- Page dashboard (intégration des composants).

### Sprint 5 — Finalisation
- Tests, Debug, UI polish (en collaboration).

## Planning Hebdomadaire (Exemple)
- **Jour 1** : Layout
- **Jour 2** : Clients
- **Jour 3** : Produits
- **Jour 4** : Factures + Stock
