# Dev 3 — Analytics & Dashboard Engineer

## Responsabilité
Intelligence métier.

## Missions
- **Tableaux de bord** :
  - Fiscal
  - Ventes
  - Stock
- **Requêtes analytiques** (LINQ avancé).
- **Intégration graphiques** (Chart.js / ApexCharts).
- **Filtres dynamiques**.
- **Gestion des permissions** : Accès limité aux stats sensibles (Admin vs Standard).
- **Data Resiliency** : Gestion des données manquantes/nulles dans les graphiques.

## Livrables
- Dashboard complet.
- Graphiques interactifs.
- Indicateurs fiables.
- Dashboard sécurisé par rôle.
- Graphiques avec gestion d'erreurs (empty/null).

## Organisation Agile (Sprints)

### Sprint 0 — Setup
- Choisir lib graphique.
- Définir KPIs.

### Sprint 1 — Core CRUD
- Préparer modèles stats.
- **Logique d'exclusion des données supprimées (Soft Delete)**.

### Sprint 2 — Facturation
- CA total.
- TVA totale.

### Sprint 3 — Stock
- Indicateurs stock :
  - Rupture
  - Alerte

### Sprint 4 — Dashboard
- Graphiques :
  - CA par période
  - TVA par taux
  - Évolution du stock

### Sprint 5 — Finalisation
- Tests, Debug, UI polish (en collaboration).

## Planning Hebdomadaire (Exemple)
- **Jour 1** : KPIs
- **Jour 2** : LINQ
- **Jour 3** : Graphiques
- **Jour 4** : Dashboard
